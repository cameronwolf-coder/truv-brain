# Cold Email Angle Rotation Library

Each 45-day wave cycle uses a different angle so the same contacts never receive the same pitch twice. Angles rotate in order; the wave scheduler tracks which angle each contact received last via `outreach_wave_angle` in HubSpot.

---

---

## Usage

The wave scheduler (`outreach_intel/wave_scheduler.py`) references these angles by their ID:

```bash
python -m outreach_intel.cli wave-build --size 2500 --angle "encompass-integration"
python -m outreach_intel.cli wave-build --size 2500 --angle "pre-closing-voe"
```

The `outreach_wave_angle` HubSpot property tracks which angle each contact received, ensuring they get a different one next cycle.
