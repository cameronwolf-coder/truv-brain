UTM → Event

1. Read UTM

Pull the contact's utm_campaign property from the enrollment event.

2. Search Events

Query Event custom objects for a record whose utm_campaign matches.

3. Associate

Create a USER_DEFINED association between the contact and the matched event record.

4. Confirm

Output event name, contact ID, and association status for downstream workflow steps.

Under the Hood

Technical details

Custom Code Action

Node.js 20.x using axios for direct HubSpot REST API calls. No SDK overhead.

Event Object Type

Custom object 2-56342751 with utm_campaign as the lookup key.

Auth & Scopes

AGENTTEST_TOKEN secret with crm.objects.custom.read and crm.objects.custom.write.

// Step 4: Dynamic UTM → Event association const searchRes = await axios.post( `/crm/v3/objects/${eventObjectType}/search`, { filterGroups: [{ filters: [{ propertyName: "utm_campaign", operator: "EQ", value: utmCampaign }] }]} ); const eventId = searchRes.data.results[0].id; // v4 Associations API — USER_DEFINED await axios.put( `/crm/v4/objects/contacts/${contactId} /associations/${eventObjectType}/${eventId}`, [{ associationCategory: "USER_DEFINED", associationTypeId: 40 }] );
