What is Agno?

The framework GTM teams use
to run AI agents

Agno is an open-source Python framework (formerly Phidata) for building AI agents that can use tools, reason over data, and work together in teams. It's becoming the standard for GTM signal detection.

🤖

Agents

An AI model (Claude) wrapped with instructions and tools. It reasons about what to do, calls functions, and returns structured results.

🔧

Tools

Plain Python functions the agent can call. Search the web, query HubSpot, check for job changes. Claude decides when to use each one.

👥

Teams

Multiple specialized agents working together. A leader delegates tasks, members gather signals in parallel, results get synthesized.

pip install agno Open source Model-agnostic Claude native support

Build effort by phase

Half day
  • Wrap scorer output in Agno agent
  • Claude prompt for reasoning
  • New CLI command

v2: Signal Enricher
1-2 days
  • Define tool functions for Firecrawl
  • Job change search tool
  • Company news search tool

v3: Multi-Agent
3-5 days
  • Agno Team with coordinate mode
  • 3 specialist agents + scorer
  • Parallel execution pipeline
  • Persistence + session state

Key technical decisions

  • Model: claude-sonnet-4 (fast, cheap, tool-capable)
  • Framework: Agno (pip install, model-agnostic, Teams built in)
  • Tools: Plain Python functions with docstrings
  • Output: Structured JSON → Slack / CLI / Pipedream
