In Action

Two automations, built this week

✉

Email Activity Tracking

Every email we send through SendGrid is now automatically logged on the contact's HubSpot timeline — opens, clicks, and deliveries.

SendGrid event fires → Find contact in HubSpot → Log email engagement
Result: Full email visibility on every contact record — no manual logging
📅

Event → Project Sync

When a marketing event is created or updated in HubSpot, a matching project is automatically created in Linear with the right status and dates.

HubSpot event triggers → Fetch event details → Create/update Linear project
Result: Every event has a project in Linear automatically — no duplicate work
