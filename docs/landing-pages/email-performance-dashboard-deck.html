The Problem

We had a blind spot.

"We could trigger emails through Knock, but had zero visibility into what happened after — did they land? Did anyone open them? Were people clicking?"
  • Knock triggers the send — but doesn't track engagement
  • SendGrid has the data — but it's siloed in their dashboard
  • No way to tie a Knock workflow to actual opens and clicks
  • Reporting meant manually pulling CSVs from two platforms

The Solution

A real-time dashboard that connects
every send to every open.

Automatic attribution across Knock workflows and SendGrid engagement — no manual work.

How It Works

The data flow

🔔 Knock Triggers workflow
→
✉️ SendGrid Sends email
→
📡 Webhook Events fire back
→
💾 Database Stored in real-time
→
📊 Dashboard Instant insights

Every open, click, bounce, and delivery is captured automatically — tagged back to the exact Knock workflow that triggered the send.
