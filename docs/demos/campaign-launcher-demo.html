Step 4 — Design the Email

A branded email, built and uploaded automatically

The AI builds a fully branded email using Truv's design system — correct fonts, colors, logo, mobile-responsive layout, unsubscribe link, and footer. It then uploads it to SendGrid as a ready-to-use template.

  • 🎨 On-brand — Gilroy font, Truv blue, dark hero gradient
  • 📱 Mobile-ready — responsive on every device
  • 👤 Personalized — greets each recipient by first name
  • ✅ Compliant — unsubscribe link included automatically

Emails go out in batches, not all at once

For large audiences, the system sends in controlled batches with pauses between them. This protects deliverability and prevents emails from landing in spam. You can always preview the plan before any emails go out.
