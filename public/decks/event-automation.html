The Challenge

Too many tools, not enough visibility

Event execution relied on manual coordination across disconnected systems.

📝

Manual task creation

Every webinar required manually creating ~31 tasks across multiple tools. Hours of setup per event.

🔀

Disconnected systems

HubSpot for CRM tasks, Linear for project management, spreadsheets for tracking. No single source of truth.

🚫

Dropped balls

Without a centralized checklist, playbook steps got missed. No visibility into who owned what.

How It Works

From event to execution in seconds

1

Event created in HubSpot

Melissa adds a new Webinar with dates and details.

2

Playbook tasks pulled from Notion

31 tasks auto-fetched covering every step from landing pages to post-event follow-up.

3

Tasks assigned with smart due dates

Assigned to the right person with due dates relative to the event.

4

Everything mirrors to Linear

Every task becomes a Linear Issue with matching assignees and due dates.

Pipedream automation: webhook → fetch event → create tasks → sync to Linear
