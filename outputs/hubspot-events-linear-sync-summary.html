Pipedream Workflow

Three-step processing pipeline

↗ Open in Pipedream 📁 GTM Automation Project
1
HTTP Trigger Receives POST from HubSpot webhook at eo6flf4pdz0kt0f.m.pipedream.net
2
Fetch Event Details Calls HubSpot API with OAuth to get full event record — name, dates, location, URL, status, type
3
Sync to Linear Searches for existing [MKTG-GROWTH] project by name. Creates new or updates existing via GraphQL API.
Field Mapping

HubSpot → Linear translation

HubSpot Event Linear Project
event_name → name with [MKTG-GROWTH] prefix
event_start_date → startDate
event_end_date → targetDate
event_location → summary (name — location)
event_status → state (planned / started / completed)
event_type + event_url → description (rich markdown)
